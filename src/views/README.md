# TCIC Device Check - Views

此目录包含 SDK 的页面级视图组件。

## 文件说明

- `MainContainer.vue` - 主容器组件（负责最外层布局和样式隔离）
- `DeviceDetect.vue` - 设备检测核心业务组件（包含侧边导航和内容区）
